ETAPA 2:
Creame la estructura para un componente funcional en React llamado MascotaCard.jsx que reciba por props de forma desestructurada: nombre, raza, edad, especie, descripcion, caracteristicas y adopcionUrgente. 
Aplica un objeto de configuración para alternar estilos en línea (colores y fondos) según la especie ('Perro', 'Gato', 'Otro'). Incluye la validación de tipos con PropTypes (definiendo caracteristicas como un array de strings) y asigna valores por defecto robustos para cada prop. 
Asegúrate de renderizar todo como contenido JSX plano y limpio, sin métodos inseguros ni datos quemados.

ETAPA 3:
Necesito crear el componente ListaMascotas.jsx que reciba un array de objetos llamado mascotas mediante props. 
Genera el renderizado iterativo usando el método .map() para devolver un componente <MascotaCard /> por cada elemento. 
Es mandatorio que le asignes la propiedad única key utilizando el id explícito del objeto de la mascota, evitando el uso del índice del array. 
Incluye su respectivo PropTypes.arrayOf con la forma (PropTypes.shape) del objeto.

ETAPA 4:
Creame el componente FiltroEspecie.jsx que renderice botones para las opciones ['Todas', 'Perro', 'Gato', 'Otro'] y maneje el evento a través de un callback onChange. 
Adicionalmente, en App.jsx implementa dos estados con useState: uno para la especie seleccionada y otro para la barra de búsqueda por nombre. 
El input de búsqueda debe ser completamente controlado. Sugiéreme una función pura de normalización que limpie espacios en blanco (.trim()) 
y limite de forma estricta la entrada a un máximo de 40 caracteres (.slice(0, 40)) para mitigar ingresos maliciosos. 
Finalmente, combina ambos filtros sobre la lista original mediante .filter() de JavaScript.
